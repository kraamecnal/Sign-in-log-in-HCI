A simple Sign in/log in website from the course HCI

Technology stacks:
HTML: Defines the structure and elements of the trash bin interface.
CSS: Styles the UI components, layouts, animations, and responsiveness.
JavaScript: Manages interactive behaviors like sensor status updates, alerts, and simulated automation features.

Running the Prototype Locally:
Step 1: Download or clone the project repository to your local machine (Source file).
Step 2: Open the 'Source file' folder (preferably in VSCode).
Step 3: Open website.html into your prefer live server or browser.
Step 4: Interact with the prototype interface to test features such as automatic lid opening, sensor status, and alerts simulated using JavaScript.

note: html, css, and js should be in a single file before running or opening in the live server.
